# NBA Shot Quality Project

This project predicts basketball shot quality using contextual features such as shot clock, dribbles, touch time, shot distance, and defender distance.

## Dataset
Primary dataset: NBA Shot Logs (Kaggle)

## Week 1
- Created project repository
- Imported raw shot log data
- Cleaned missing and malformed values
- Converted game clock to numeric seconds
- Built binary target variable for made/missed shots
- Saved cleaned dataset for modeling
